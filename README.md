# Portafolio Web Personal  
## Evidencia de Desarrollo Frontend

---

## 1. Introducción

El presente proyecto corresponde al desarrollo de un portafolio web personal, diseñado con el objetivo de aplicar los conocimientos adquiridos en HTML5, CSS3 y JavaScript, integrando principios de diseño moderno, estructura semántica y animaciones interactivas.

El sitio fue desarrollado bajo una temática visual oscura con acentos en tonos azules, buscando transmitir profesionalismo, modernidad y estructura organizada.

---

## 2. Objetivos del Proyecto

### Objetivo General
Desarrollar un portafolio web responsive que permita presentar el perfil profesional, habilidades técnicas y proyectos académicos.

### Objetivos Específicos
- Aplicar estructura semántica en HTML5.
- Implementar diseño moderno con CSS3.
- Integrar animaciones mediante JavaScript.
- Utilizar la API Intersection Observer.
- Incorporar efectos visuales con la librería tsParticles.
- Diseñar una interfaz adaptable a diferentes dispositivos.

---

## 3. Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Intersection Observer API
- tsParticles
- Font Awesome

## 4. Estructura del Proyecto

## 5. Descripción de Componentes

### 5.1 Barra Lateral (Sidebar)
- Navegación vertical fija.
- Expansión dinámica mediante JavaScript.
- Mejora de experiencia de usuario.


### 5.2 Animaciones de Scroll
Implementadas mediante Intersection Observer para activar clases CSS cuando los elementos entran en el viewport.

### 5.3 Fondo Animado
Se utilizó la librería tsParticles para crear un efecto dinámico de partículas en el fondo.


### 5.4 Sección de Proyectos
Presentación estructurada de trabajos académicos y personales, en esta parte se separa por secciones cada tematica, de este modo se mira mas ordenada. 




### 5.5 Sección Multimedia
Integración de un video incrustado mediante iframe para reforzar la identidad visual del portafolio.
